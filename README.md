# 🌐 Shivprasad Bansode — Portfolio Website

This is my personal developer portfolio — a single-page React application I built to showcase my work as a final-year B.E. (Electronics & Telecommunication) graduate transitioning into Java Full Stack development. It pulls together my technical skills, real-world experience, academic background, and hands-on projects into one clean, animated site.

The site covers five skill categories (Frontend, Backend, Languages, Dev Tools, and AI & OS), my professional experience as a Jr. Software Developer at Creazione Software and an App Development Intern at Suvidha Foundation, my full academic timeline from SSC through my B.E., and project deep-dives including a Shopping Cart app and an Image Background Remover. A working contact form (via EmailJS) lets visitors reach out directly.

## 🧑‍💻 Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/shivbansode369/Shiv-Portfolio.git
cd Shiv-Portfolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the project root (see `.env.example` for the required keys):

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id_here
VITE_EMAILJS_TEMPLATE_ID=your_template_id_here
VITE_EMAILJS_PUBLIC_KEY=your_public_key_here
```

Get these values from your [EmailJS dashboard](https://dashboard.emailjs.com/).

### 4. Run the App Locally

```bash
npm run dev
```

### 5. Build for Production

```bash
npm run build
```

## ☁️ Deploying on Vercel

1. Push this repo to GitHub.
2. Import the repo into [Vercel](https://vercel.com/).
3. Add the same three environment variables (`VITE_EMAILJS_SERVICE_ID`, `VITE_EMAILJS_TEMPLATE_ID`, `VITE_EMAILJS_PUBLIC_KEY`) under **Project Settings → Environment Variables**.
4. Deploy 🚀
